任务一 Task1.java:
   先计算出不同用户在不同时段在不同基站停留的时间.

   输入格式 ：
    * 输入参数 < input path > < output path > < date > < timepoint >
    * 参数示例：
        $ContentRoot$/input/data/
   	$ContentRoot$/output
   	2021-07-01
   	07-09-17-24

    * 意味着以“ $ContentRoot$/input/data/”为输入，"$ContentRoot$/output"为输出，
    指定计算2021-07-01的数据，并分为00-07，07-09, 09-17, 17-24四个时段

   输出格式:   用户IMSI|基站|时间段|时长(分钟)
   示例:
     0000000001|00000032|00-07|139.9
     0000000001|00000163|00-07|10.766666
     0000000001|00000030|00-07|2.3166666
